# Awesome Learn

A curated collection of learning resources for various technologies and concepts. This book organizes markdown-based guides across different technology domains into a navigable, book-style format.

## Topics Covered

- **Commerce Systems** — E-commerce platforms, payment systems, and online retail technologies
- **Kubernetes** — Container orchestration, cloud-native technologies, and the K8s ecosystem
- **Observability** — Monitoring, logging, tracing, and system observability

## How to Use This Book

Each section is organized with numbered chapters that build on each other. Start with the overview in each section, then follow the learning path or dive into specific topics as needed.

- **New to a topic?** Start with the Overview (00) and follow the Learning Path
- **Looking for something specific?** Use the sidebar to jump to the relevant chapter
- **Want a structured approach?** Follow the Learning Path in each section for a guided curriculum

## Reading Formats

This book is published in multiple formats so you can read it wherever you like:

| Format | Description |
|--------|-------------|
| **HTML** | Interactive web version with search, dark mode, and collapsible sidebar. The default format hosted online. |
| **PDF** | Printable single-file export generated with [mdbook-pdf](https://github.com/HollowMan6/mdbook-pdf) via headless Chromium. Download from the CI artifacts. |
| **EPUB** | E-reader friendly format (Kindle, Kobo, etc.) — can be added via the [mdbook-epub](https://github.com/Michael-F-Bryan/mdbook-epub) backend. |

## Themes & Dark Mode

The HTML version ships with five built-in themes you can switch using the **paintbrush icon** (🎨) in the top toolbar:

| Theme | Style |
|-------|-------|
| `light` | Default light |
| `rust` | Warm light |
| `coal` | Dark |
| `navy` | Dark blue *(default for this book)* |
| `ayu` | Modern dark *(preferred when your OS uses dark mode)* |

Your theme choice is remembered between visits via `localStorage`.
